# Waste Classification using MobileNetV2

This project classifies waste images using a deep learning model (MobileNetV2).

## Features
- Image classification of waste
- Grad-CAM heatmap visualization
- Highlights important regions used by the model

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV

## Description
The model predicts the type of waste from input images. Grad-CAM is used to visualize which parts of the image influence the prediction, improving interpretability.
